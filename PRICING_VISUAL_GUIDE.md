## Color Scheme

### Popular Tier (Growing Farm)
- **Header**: Linear gradient (Blue 500 → Blue 700)
- **Background**: Gradient (Blue 50 → Indigo 50)
- **Border**: Blue 500
- **Shadow**: Blue 100

### Other Tiers
- **Header**: Gradient (Gray 50 → Slate 50)
- **Border**: Transparent (2px)
- **Shadow**: Light gray

---

## Interactive States

### Hover (Desktop)
- Card translates up 8px (`-translate-y-2`)
- Shadow increases
- For popular tier: maintains scale effect

### Active (Mobile)
- Button shows active state
- Touch feedback with subtle scale

### Focus
- Focus ring visible
- Keyboard navigation supported

## Customization Guide

### To Change Pricing
Edit `src/app/page.tsx` → `pricingTiers` array

### To Change Colors
Edit `src/app/globals.css` → `.pricing-grid`, color values

### To Change Layout Columns
Edit `.pricing-grid` breakpoints:
- `grid-cols-1` (mobile)
- `md:grid-cols-2` (tablet)
- `lg:grid-cols-4` (desktop)

### To Change Feature List Height
Edit `.pricing-grid` section → `max-h-96` (24rem)
